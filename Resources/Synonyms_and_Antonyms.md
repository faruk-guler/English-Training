# Synonyms and Antonyms - Zıt ve Eş Anlamlı Kelimeler Sözlüğü

İngilizcede kelime dağarcığınızı genişletmenin en iyi yollarından biri, sürekli "Very (çok)" kelimesini kullanmak veya hep aynı kelimeleri tekrar etmek yerine daha zengin (ileri seviye) eş anlamlı kelimeler (Synonyms) kullanmaktır. Bu dosya, en sık kullanılan temel kelimelerin ileri seviye eş anlamlılarını ve zıt anlamlılarını (Antonyms) içerir.

## 1. Sıfatlar (Adjectives)

Sürekli aynı sıfatları kullanmak yerine bu alternatiflerle cümlenizi zenginleştirebilirsiniz.

| Temel Kelime | Türkçe Anlamı | Daha Güçlü Eş Anlamlıları (Synonyms) | Zıt Anlamlıları (Antonyms) |
| :--- | :--- | :--- | :--- |
| **Good** | İyi, güzel | Excellent, Superb, Outstanding, Marvelous | Bad, Awful, Terrible, Poor |
| **Bad** | Kötü | Terrible, Awful, Dreadful, Horrible | Good, Excellent, Wonderful |
| **Big** | Büyük | Huge, Massive, Enormous, Gigantic | Small, Tiny, Miniature, Microscopic |
| **Small** | Küçük | Tiny, Petite, Minute, Compact | Big, Huge, Enormous |
| **Happy** | Mutlu | Delighted, Joyful, Thrilled, Ecstatic | Sad, Miserable, Depressed, Unhappy |
| **Sad** | Üzgün | Miserable, Sorrowful, Heartbroken, Upset | Happy, Joyful, Cheerful |
| **Important** | Önemli | Crucial, Essential, Significant, Vital | Unimportant, Trivial, Insignificant |
| **Interesting**| İlginç | Fascinating, Captivating, Engaging, Intriguing | Boring, Dull, Uninteresting |
| **Boring** | Sıkıcı | Dull, Tedious, Monotonous, Unexciting | Interesting, Exciting, Fascinating |
| **Beautiful** | Güzel | Gorgeous, Stunning, Breathtaking, Attractive | Ugly, Hideous, Unattractive |
| **Ugly** | Çirkin | Hideous, Unattractive, Grotesque | Beautiful, Gorgeous, Attractive |
| **Smart** | Zeki | Clever, Brilliant, Intelligent, Bright | Stupid, Foolish, Ignorant |
| **Tired** | Yorgun | Exhausted, Drained, Fatigued, Weary | Energetic, Lively, Refreshed |
| **Angry** | Sinirli | Furious, Enraged, Indignant, Irritated | Calm, Peaceful, Relaxed |
| **Fast** | Hızlı | Quick, Rapid, Swift, Hasty | Slow, Sluggish, Leisurely |
| **Hard** | Zor | Difficult, Challenging, Tough, Complex | Easy, Simple, Effortless |

## 2. Fiiller (Verbs)

Eylem bildiren kelimelerde çeşitlilik yaratmak, konuşma ve yazma (writing) becerisini çok yukarı taşır.

| Temel Kelime | Türkçe Anlamı | Daha Güçlü Eş Anlamlıları (Synonyms) | Zıt Anlamlıları (Antonyms) |
| :--- | :--- | :--- | :--- |
| **Like** | Sevmek, hoşlanmak | Enjoy, Admire, Appreciate, Adore | Dislike, Hate, Loathe, Detest |
| **Hate** | Nefret etmek | Loathe, Detest, Despise, Abhor | Love, Adore, Like |
| **Help** | Yardım etmek | Assist, Support, Aid, Guide | Hinder, Prevent, Obstruct |
| **Start** | Başlamak | Begin, Initiate, Commence, Launch | Finish, End, Conclude |
| **Finish** | Bitirmek | Complete, Conclude, End, Terminate | Start, Begin, Initiate |
| **Show** | Göstermek | Demonstrate, Reveal, Display, Exhibit | Hide, Conceal, Cover |
| **Use** | Kullanmak | Utilize, Employ, Operate, Consume | Misuse, Neglect, Ignore |
| **Think** | Düşünmek | Believe, Consider, Contemplate, Ponder | Ignore, Disregard |
| **Change** | Değiştirmek | Alter, Modify, Transform, Adjust | Remain, Keep, Maintain |
| **Stop** | Durdurmak | Cease, Halt, Pause, Terminate | Continue, Proceed, Go |

*İpucu: Speaking veya Writing (IELTS/TOEFL) sınavlarında "very big" demek yerine "enormous", "very bad" demek yerine "awful" kelimelerini kullanmak puanınızı doğrudan artırır.*
